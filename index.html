Simplistic Design

This is one of the projects that I am very proud of. Simplistic Design is an CSS and JavaScript library to make the process of creating a web site a lot easier. Simplistic Design is an ongoing project, constantly evolving with web trends. Simplistic Design is, technically, customizable, it just won't be easy in the either compacted or thousand-line long CSS stylesheets. View project here
